Place the homepage logo in this folder and keep the filename:

`medisignes-home-logo.png`

The home page loads this exact file from:

`assets/branding/medisignes-home-logo.png`

To replace the current placeholder:

1. Export your logo as a transparent PNG.
2. Replace `medisignes-home-logo.png` with your own file.
3. Keep the same filename so no code change is needed.

Recommended:

- Wide logo format
- Transparent background
- Around 1000px wide or more for crisp rendering
